Spinner Plugin Demo

The Spinner plugin allows your app to programmatically show or hide a spinner. The spinner has a platform specific appearance, giving your app a nice native touch.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/WinPhone - App package.

Showing the spinner

You can tap the screen (or the backbutton on Android) to hide the spinner.

Persistent spinner

Tap to hide is disabled, so hide it programmatically. This demo will hide it after 5 seconds.

Android-only features

iOS and Windows Phone have only one line of text. They will show the 'title' only if no 'message' is passed.
